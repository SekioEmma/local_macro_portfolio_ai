# Local Macro Portfolio AI

Local Macro Portfolio AI 是一个本地个人投资研究与资产配置辅助系统，用于帮助个人在本地整理持仓、目标配置和定投规则，并计算账户状态、资产配置偏离与预算压力。

本项目不是自动交易系统，不预测市场，不保证收益。它面向学习型长期投资实践，第一版只做基础数据读取与资产配置分析，为后续接入公开数据源、RAG 和本地模型做准备。

## 不做什么

- 不预测市场
- 不保证收益
- 不自动交易
- 不鼓励杠杆
- 不鼓励频繁交易

## 第一版 MVP

- 读取持仓 CSV
- 读取用户配置
- 计算账户状态
- 计算资产配置偏离

## 后续阶段

- 接入公开数据源
- 建立 RAG 知识库
- 接入本地模型
- 探索微调流程
- 适配 iPad 部署
